# Tool Specifications

This document will be filled in during Phase 4 with the exact JSON schemas
for each ADK tool function, including argument types, return shapes, and
usage examples.

## Tools planned

| Tool function | Purpose |
|---------------|---------|
| `get_overview` | Return Level-0 overview tile + counts |
| `get_tile_image` | Return a specific tile at a given zoom level |
| `lookup_component` | Fetch properties of a single component by ID |
| `trace_connection` | Find the path between two components |
| `get_text_in_region` | Get OCR text within a normalized bbox |
| `extract_bom` | Return the full bill of materials |

See `docs/implementation-plan.md` Phase 4 for full specification.
